# Task placement Plugin

This is an Obsidian plugin that moves tasks to the bottom/top of the editor based on their completion status

## How to Build for development

```shell
npm run dev
```

## How to Test

```shell
npm run test
```

Or test watch

```shell
npm run test:watch
```
